# 1. 클라이언트 사이드 저장소

현대 웹 브라우저들은 사용자의 동의 하에 사용자 클라이언트 측에 정보를 저장하고 필요할 시 검색할 수 있는 방법을 제공한다. 

이런 정보를 클라이언트 저장소에 저장하면 사용자가 페이지를 떠나도 로그인이나 설정 등을 유지할 수 있다. 흔히 액세스 토큰을 세션 스토리지에 저장하느니 하는 말을 통해서 들어보았을 수도 있다.
